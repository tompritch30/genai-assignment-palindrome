## WHAT IS A PROPERTY SALE (OR PROPERTY AS WEALTH)?

Property transactions represent **real estate as a source of wealth** - either through selling property for cash or holding property that constitutes equity. For KYC/AML purposes, we need to understand both the liquidity event (sale) and the original acquisition (how they got the property).

**Why this matters for compliance:**
- Property is often people's largest asset class
- The acquisition chain is crucial: How was the original purchase funded?
- Property can be sold, retained, or inherited - all represent wealth

**The full property chain:**
1. **Acquisition**: How did they get the property? (Purchased with savings, inherited, gifted, bought with mortgage)
2. **Holding**: How long did they own it? Was it rented out?
3. **Disposition**: Sold for proceeds, or still held as equity

**Property as wealth even when not sold:**
Properties contribute to net worth whether or not they've been liquidated. A retained property worth £850k is still wealth - we just note "N/A - Property retained" for sale fields while capturing the acquisition and current value.

**Common scenarios:**
- Buy-to-let investment sold for profit
- Primary residence sold when relocating
- Inherited property sold by beneficiary
- Property retained as equity component of net worth

## ENRICHED FIELD FORMAT GUIDANCE
Provide CONTEXT and DETAIL in field values to make compliance review easier:

For property_address:
- Include area and city/region: "Hampstead, North London", "Surrey countryside"
- International: "South of France", "Costa del Sol, Spain"
- If partial: "South London (exact address not stated)"

For property_type:
- ALWAYS include classification AND description: "Residential - Primary home (four-bedroom Victorian house)"
- Categories: "Residential - Primary home", "Residential - Buy-to-let investment", "Residential - Holiday home", "Commercial"
- Use formal property classifications, not informal descriptions (e.g., "Residential - Primary home" not "old family home")
- Examples:
  - "Residential - Primary home (detached house)"
  - "Buy-to-let investment"
  - "Townhouse - Primary residence"
  - "Commercial - Office building"

For original_acquisition_method:
- Include SOURCE of funds: "Purchased with savings and mortgage"
- Include context: "Purchased using employment savings", "Purchased using divorce settlement funds"
- For joint: "Purchased jointly"
- If inherited: "Inherited from father"

For original_acquisition_date:
- Standard format: "2005", "March 1998"

For original_purchase_price:
- Use full numbers: "£1,100,000" not "£1.1 million"
- If inherited, set to null (not applicable)

For sale_date:
- Standard format: "March 2021", "August 2022"

For sale_proceeds:
- Include NET vs GROSS context: "£620,000 (net after mortgage repayment and costs; gross £925,000)"
- If just net: "£450,000 (net of fees and mortgage)"
- If mentioned as net: "£365,000 (net proceeds after costs; gross £385,000)"
- Use full numbers: "£4,800,000" not "£4.8 million"
- For foreign: "€2,200,000 (~£1,900,000)"

Return a list of SaleOfPropertyFields objects, one for each property sale found.
